# 🌱 AgroViva - Inteligência, Gestão e Sustentabilidade

[cite_start]O **AgroViva** é uma aplicação web interativa voltada para o setor agrícola sustentável, desenvolvida para a **Subcategoria 3: Programação Front-End** do Concurso Agrinho 2026[cite: 242].

## 🎯 Objetivo e Tema Agrinho
[cite_start]Este projeto foi construído para atender estritamente ao tema central do concurso: **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**[cite: 243]. O objetivo do AgroViva é demonstrar como a tecnologia pode auxiliar o produtor rural na tomada de decisões precisas. O sistema oferece simuladores interativos focados em gestão financeira, mercado de carbono, previsões climáticas e diagnósticos de estufas.

## 🚀 Tecnologias Utilizadas
[cite_start]O projeto foi desenvolvido respeitando as diretrizes da Subcategoria 3, utilizando exclusivamente linguagens Front-End[cite: 243]:
* **HTML5:** Estruturação semântica do conteúdo e formulários interativos.
* **CSS3:** Estilização autoral, variáveis de cor, layout responsivo (Media Queries) adaptável para dispositivos móveis e animações visuais.
* [cite_start]**JavaScript (Vanilla):** Manipulação funcional do DOM, processamento matemático local das calculadoras, simulação de cotações dinâmicas e programação do minigame interativo utilizando Canvas[cite: 246, 247].

## 📖 Instruções de Uso
1.  **Acesso Inicial:** Abra o link do projeto disponibilizado na seção "About" deste repositório (GitHub Pages). Ao visualizar a tela de abertura, clique em "Entrar na Propriedade".
2.  **Navegação no Painel:** Utilize o menu de abas superior para transitar entre os diferentes módulos da fazenda inteligente (Financeiro, Carbono, Clima, etc.). O Dashboard principal reflete as mudanças do seu progresso em tempo real.
3.  **Uso dos Simuladores:** Nas abas interativas, insira os valores numéricos ou selecione as opções nos campos indicados e clique no botão de ação (ex: "Calcular Carbono" ou "Analisar"). O sistema retornará diagnósticos visuais e recomendações baseadas nos dados inseridos.
4.  **AgroViva Harvest (MiniGame):** Acesse a aba "MiniGame" para jogar. Em computadores, utilize as setas **Esquerda** e **Direita** do teclado para mover o trator. Em dispositivos móveis (celulares e tablets), utilize os botões em tela para realizar a colheita. Os pontos obtidos impactam diretamente o ranking local do navegador.
